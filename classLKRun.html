Detailed Description

If output file name is not set, output file name will be named as below. In general: [run_name]_[run_id(4-digits)].[lilak_version].root; If tag exist: [run_name]_[run_id(4-digits)].[user-tag].[lilak_version].root; If splited: [run_name]_[run_id(4-digits)]_s[split_no].[user-tag].[lilak_version].root;
