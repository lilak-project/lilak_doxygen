Detailed Description

Orthogonal Distance Regression (ODR) Fitter.

Orthogonal distance regression plane is found by minimizing f(X) (sum of perpendicular distance from point to the plane). The process is done by solving the eigen value equation of matrix M that is made from f(X). The ODR plane choosen with the smallest eigen value.

Orthogonal distance regression line is found by minimizing the sum of perpendicular distance from point to the line. The ODR line is choosen with the largest eigen value of matrix M.
