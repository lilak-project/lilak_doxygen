double &  continuousLength,
double  distCut = 25 
)
virtualinherited

Check continuity of the track. Hit array must be filled. Returns ratio of the continuous region. (-1 if less than 2 hits)

If difference between "length from hits" is smaller than distCut, (> pad-diagonal, < 2*pad-row) the region is considered to be connected.

The total length calculation is different from TrackLength(), because TrackLength() use only the head/tail alpha angle information but this method calculate length directly from the hit using Map(...) method.

Parameters
continuousLengthcalculated length of continuos region.
totalLengthcalculated total length of the track.
