)
virtualinherited

Implementation recommanded. Set poseReco to global position using pad position, and time-bin tb. Pad position must be set from Init(). Conversion from time-bin (tb) to position in time-axis (post) is calculated by : post = fTbToLength * tb + fPosition [fTbToLength] is tb to position conversion factor with direction and [fPosition] is detector plane position in time-axis.

Parameters
pad[in]channel (pad)
tb[in]Time-bin
posReco[out]Reconstructed position of electron(?) before drift.
driftLength[out]Drift length of the electron from the reconstructed position.
